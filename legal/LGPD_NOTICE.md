## 2. Base Legal para o Tratamento

O tratamento de dados realizado pelo Software baseia-se no:

- **Art. 7º, I** — consentimento do titular (você opta por utilizar o Software e escolher o provedor de IA);

- **Art. 7º, V** — execução de contrato (uso do Software conforme os Termos de Uso).

## 3. Dados Tratados

O Software processa exclusivamente os arquivos PDF que você seleciona ativamente. Os tipos de dados extraídos podem incluir:

- Nome do documento
- Tipo de documento (holerite, nota fiscal, contrato, etc.)
- Datas (emissão, competência, vencimento)
- Valores monetários
- Nome do funcionário/empresa (quando presente no documento)

## 4. Finalidade do Tratamento

Única e exclusivamente para organizar, renomear e classificar os arquivos PDF conforme os metadados extraídos.

## 5. Compartilhamento

Os dados são compartilhados apenas com o provedor de IA explicitamente selecionado por você na interface do Software. Consulte a política de privacidade do provedor escolhido para mais detalhes.

## 6. Retenção e Exclusão

Os metadados extraídos são armazenados apenas durante a sessão de uso. Os arquivos gerados ficam sob seu controle exclusivo em seu sistema de arquivos. A exclusão pode ser feita a qualquer momento removendo os arquivos gerados manualmente.

## 7. Seus Direitos

Nos termos da LGPD, você pode:
1. Confirmar a existência de tratamento de dados;
2. Acessar os dados tratados;
3. Corrigir dados incompletos, inexatos ou desatualizados;
4. Solicitar a anonimização, bloqueio ou eliminação de dados;
5. Revogar o consentimento a qualquer momento.

Para exercer seus direitos, entre em contato pelo GitHub.
